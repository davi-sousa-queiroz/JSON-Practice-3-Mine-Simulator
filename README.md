# ⛏️ Mining Simulator (JSON Practice 3) (April 5th-6th 2026)

A simple terminal-based mining game built with Python.
Mine resources, manage your inventory, and sell items for coins!

⸻

## 🎮 Features
	•	⛏️ Mine for resources (Coal, Iron, Gold, Diamonds)
	•	🎒 Inventory system
	•	💾 Save & load system using JSON
	•	🛒 Shop system to sell items
	•	🎲 Randomized mining outcomes

⸻

## 🧠 What I Learned
	•	Working with classes and organizing code across files
	•	Handling JSON data for saving and loading game progress
	•	Building a game loop with user input
	•	Debugging common errors (file handling, dictionaries, etc.)
	•	Structuring a small project like a real program

⸻

## 📁 Project Structure
  • main.py → Runs the game loop and menu
  • mine.py → Handles mining logic and inventory
  • file.py → Handles saving and loading JSON data
  • economy.py → Handles selling items

⸻

## ▶️ How to Run
	1.	Make sure you have Python installed
	2.	Run the game: python main.py

## 💾 Save System
	•	Game data is stored in player_data.json
	•	Automatically creates the file if it doesn’t exist
	•	Saves progress when quitting the game

⸻

## 🛒 Shop

Sell items for coins:
	•	Coal → 1 coin
	•	Iron Ore → 5 coins
	•	Gold Ore → 10 coins
	•	Diamonds → 50 coins
